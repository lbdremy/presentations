Node.js kesado?




Une plateforme de développement pour écrire des programmes réseaux en javascript.

Qu'est ce que Node.js a de spécial?




  • Basé sur une boucle événementielle
  • Les opérations effectuées sur le disque, le réseau ou une BDD sont asynchrones, non bloquantes
  • Votre programme Javascript tourne dans un seul thread
  • Fournie un ensemble de bibliothèques bas niveau de nature asynchrone pour compléter javascript
